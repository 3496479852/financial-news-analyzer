# 📊 Financial News Analyzer

> 基于多 Agent 协作架构的全球金融市场智能新闻分析系统

---

## 项目简介

本项目面向机构投资与研究团队，通过 AI 驱动的多 Agent 流水线，对全球多渠道金融新闻进行实时聚合、分类、情绪分析与影响评估，辅助股票及债券市场的投资决策。

---

## 核心痛点

金融市场每日产生海量多语言新闻，研究员需手动筛选并判断其对持仓标的的影响，效率低下且存在信息遗漏风险。在美联储议息、非农数据等关键节点，信息量激增数倍，人工处理已无法满足实时决策需求。本系统旨在消除信息获取与投资判断之间的时滞。

---

## 系统架构（多 Agent 协作）

```
[ 新闻源 ]
彭博 / 路透 / CNBC / ...
        │
        ▼
┌─────────────────┐
│   采集 Agent     │  实时抓取、去重、语言识别、初步分类
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   分析 Agent     │  Chain-of-Thought 链式推理
│                 │  事件识别 → 标的判断 → 影响评估
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   情绪 Agent     │  宏观 / 行业 / 个股三维情绪评分
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   校验 Agent     │  高影响事件二次推理验证，过滤误报
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│   汇总 Agent     │  生成每日简报 & 实时预警推送
└─────────────────┘
```

---

## 主要功能

- **实时新闻摘要**：对全球多语言新闻进行结构化摘要，节省研究员阅读时间
- **市场情绪分析**：股票、债券相关新闻多维度情绪评分，辅助仓位判断
- **事件影响评估**：识别宏观事件（政策、财报、地缘）对标的资产的潜在影响
- **批量文本处理**：每日定时批量拉取新闻并分类标注
